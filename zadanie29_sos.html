1. W jaki sposób można określić właściwość danego obiektu?

Właściwość obiektu w JavaScript można określić na dwa sposoby:

  • Za pomocą notacji kropkowej:
    obiekt.właściwość
    Przykład: let samochód = { marka: "Toyota" };
    console.log(samochód.marka);
  • Za pomocą notacji nawiasowej:
    obiekt["właściwość"]
    Przykład: console.log(samochód["marka"])

2. W jaki sposób można odwołać się do pól (właściwości) i metod?

  • Notacja kropkowa: obiekt.pole, obiekt.metoda()
  • Notacja nawiasowa: obiekt["pole"], obiekt["metoda"]()
let osoba = { imię: "Jan", przywitaj: function() { console.log("Cześć!"); } };
osoba.imię;
osoba.przywitaj();

3. Wymień cechy charakterystyczne konstruktora

  • Konstruktor jest specjalną funkcją do tworzenia i inicjalizacji obiektów.
  • Nazwa konstruktora zaczyna się wielką literą.
  • Wywołuje się go przy pomocy słowa kluczowego new.
  • Może przyjmować parametry definiujące początkowy stan obiektu.
  • Nie zwraca jawnie wartości – automatycznie zwraca nowy obiekt.

4. Konstruowanie obiektu – przykład

function Samochód(marka, model, rok) {
    this.marka = marka;
    this.model = model;
    this.rok = rok;
    this.opis = function() {
        return `${this.marka} ${this.model} (${this.rok})`;
    }
}
let auto = new Samochód("Toyota", "Corolla", 2020);
console.log(auto.opis());

5. Znaczenie słowa this. Jak oddzielamy pola oraz metody?

this odnosi się do aktualnego obiektu, w którym jest wywoływane.

this.pole = wartość;
this.metoda = function() { ... };

6. Konstruktor w JS – przykład i opis

function Osoba(imię, wiek) {
    this.imię = imię;
    this.wiek = wiek;
    this.przywitaj = function() {
        console.log(`Cześć, jestem ${this.imię}`);
    };
}
let osoba1 = new Osoba("Anna", 25);
osoba1.przywitaj();

7. Znaczenie słowa kluczowego new wraz z przykładem

Słowo new tworzy nową instancję obiektu z konstruktora i ustawia this wewnątrz konstruktora na nowy obiekt.

let auto = new Samochód("Honda", "Civic", 2018);

8. Znaczenie słowa kluczowego prototype wraz z przykładem i opisem

prototype pozwala dodawać właściwości i metody do wszystkich instancji klasy lub konstruktora bez modyfikowania samego konstruktora.

Samochód.prototype.kolor = "niebieski";
let auto2 = new Samochód("Ford", "Focus", 2019);
console.log(auto2.kolor);
