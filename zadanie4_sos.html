1) Kiedy powstał JavaScript (podaj datę)
JavaScript powstał w 1995 roku.

2) Dlaczego jest to język skryptowy
Ponieważ kod JavaScript jest interpretowany w trakcie działania programu przez silnik JS w przeglądarce lub środowisku (np. Node.js), a nie kompilowany wcześniej do kodu maszynowego.

3) Gdzie jest wykonywany JS (klient lub serwer)
JavaScript jest wykonywany głównie po stronie klienta (w przeglądarce), ale dzięki środowisku Node.js może działać także po stronie serwera.

4) Dlaczego JS to bezpieczne narzędzie
Ponieważ działa w tzw. piaskownicy (sandbox) – nie ma bezpośredniego dostępu do plików użytkownika ani systemu operacyjnego, a jego działanie jest ograniczone przez przeglądarkę.

5) Czy jest to język obiektowy, jeśli tak to dlaczego
Tak, JavaScript jest językiem obiektowym, ponieważ obsługuje obiekty, metody i dziedziczenie. Jego model obiektowy opiera się na prototypach zamiast na klasach (choć od ES6 dostępna jest składnia klasowa).

6) Wymień zalety JS:
Zaleta1: Działa na wszystkich popularnych przeglądarkach i systemach operacyjnych.
Zaleta2: Umożliwia tworzenie dynamicznych i interaktywnych stron internetowych.
Zaleta3: Ma dużą społeczność, wiele bibliotek i frameworków wspierających rozwój.

7) Framework to:
Gotowy szkielet programistyczny (zestaw narzędzi, bibliotek i struktur), który ułatwia i przyspiesza tworzenie aplikacji według określonych wzorców.

8) Do aplikacji mobilnych można użyć Framework
Tak, istnieją frameworki pozwalające tworzyć aplikacje mobilne w JavaScript, np. React Native, Ionic czy NativeScript.

9) Aplikacje responsywne to:
Aplikacje lub strony internetowe, które automatycznie dostosowują układ i wygląd do różnych rozmiarów ekranu (telefon, tablet, komputer).

10) Przepisz linie kody pod nimi wytłumaczenie jak działa ten fragment kodu
‹input type="button" name="przycisk" value="Nowa Strona" onclick="WinOpen(' ')"›
Tworzy przycisk o nazwie „Nowa Strona”. Po jego kliknięciu wywoływana jest funkcja WinOpen(), która może otworzyć nowe okno lub stronę (adres podawany jest jako argument).

11) Przepisz linie kody pod nimi wytłumaczenie jak działa ta ten fragment kodu
window.open("obraz.html","okienko","toolbar=no,directories=no,menubar=no,height=280,width=160,top=200,left=200");
Instrukcja otwiera nowe okno przeglądarki z plikiem obraz.html i nadaje mu nazwę „okienko”. Parametry określają wygląd i pozycję okna:
toolbar=no → ukrywa pasek narzędzi
directories=no → ukrywa przyciski katalogów
menubar=no → ukrywa menu
height=280 → wysokość okna 280 pikseli
width=160 → szerokość okna 160 pikseli
top=200 → pozycja okna 200 pikseli od górnej krawędzi ekranu
left=200 → pozycja okna 200 pikseli od lewej strony ekranu

12) Opisz następującą instrukcję window.close()
Powoduje zamknięcie bieżącego okna lub karty przeglądarki, ale tylko wtedy, gdy zostało ono otwarte przez skrypt (np. przez window.open()).
